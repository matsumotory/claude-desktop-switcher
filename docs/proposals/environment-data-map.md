# 環境インスペクタ: 環境のデータの中身とサイズの表示

## 背景・本セッションの完了事項

2026-07-03 のサーベイで、分離を売るツールこそ「各環境の中身が何で、どこにあり、どれだけあるか」を見せることが分離の実感につながる、という結論を得た。既存の Claude 系切替ツールは一覧・切替までで中身の可視化を提供していない。会話履歴が環境の数だけ増えうる CSW では、ディスクの行方への答えにもなる。ユーザーは 4 案すべての実施を承認済みで、本 plan は 3 本目である (1 本目 PR #149、2 本目 PR #151)。

## ゴール

ターミナルを使わない利用者が、環境の詳細画面でその環境のデータの実体を見通せるようにする。

- 既存の「場所（この環境のデータ）」の折りたたみを拡張し、次を表示する。
  1. デスクトップ用・Claude Code 用のデータフォルダそれぞれのパス・概算サイズ・「Finder で表示」。
  2. 項目別の内訳 (SPECIFICATION §3 の 11 項目): 共有の項目はリンク先のパス、それ以外は概算サイズと最終更新。
  3. 環境全体の専有容量 (共有リンクの先は含まない)。
- 集計はフォルダを開いたときに読み込む (遅延読み込み)。

## 設計原則 (敵対検証で確定した仕様条件)

- **シンボリックリンクを決して辿らない**: サイズ集計の walk は `symlink_metadata` ベースで行い、リンク (ファイル・ディレクトリとも) は辿らずその存在だけ数える。共有項目はリンク先パスの表示のみで、既存の Claude 側は walk しない。素朴な `fs::metadata` はリンクを辿ってしまう典型的な罠のため、非追跡であることをユニットテストで固定する。
- **メタデータのみ**: 集計はファイル名の列挙と stat (サイズ・更新日時) だけで、ファイルの中身は一切読まない。
- **サインイン情報の露出は最小限**: アカウントのサインイン情報の行は「常に分離」の事実だけを示し、サイズ・更新日時・Finder 導線を出さない。
- **合計の意味を明示**: 合計は「この環境が専有している容量」と表記し、共有リンクの先の実体を含まないことを UI 文言で示す。
- **既存の Claude は集計しない**: 既定環境のデータは他アプリの実データそのものなので、詳細画面は従来どおりパス表示のみとし、サイズ集計の対象にしない。
- **性能**: 大きな環境での集計時間を実測し、UI は遅延読み込み (折りたたみを開いたときに取得) とする。
- 公開コピーで他社ツール名を出さない。ガイドの記述範囲は SPECIFICATION §3 の項目に限定する (Claude 本体の内部レイアウト一般を網羅しない)。

### 実装ステップ

1. data_map のテストを先に書き RED を確認 → 実装で GREEN (リンクを辿らないこと・中身を開かないことを固定)
2. Tauri command + UI の遅延読み込み (minimalist-ui / japanese-typography-qa / csw_product_canon のゲートを通す)
3. 大きめのファイル群 (数千ファイル) で集計時間を実測
4. docs / PRIVACY 同期
5. 敵対検証 workflow → 指摘反映 → CI green → マージ

### 検証計画

```bash
cargo fmt --check && cargo clippy --workspace --all-targets -- -D warnings
cargo build --workspace && cargo test --workspace
```

- GUI は launch の app (devInvoke モック) で ja/en の実描画確認。実機の実環境でも表示を確認する (read-only)。
- スクリーンショット: 「場所」の折りたたみは既定で閉じているため、既存スクリーンショットに変化はない (開いた状態は撮影対象に含めない)。変化が出る場合のみ再生成する。

## リスクと対応

- **集計が遅い環境**: 遅延読み込みに加え、実測で問題があれば項目別の集計を並列化または上限打ち切りにする (打ち切り時はその旨を表示する)。
- **リンク追跡の混入**: テストで固定する (上記)。

## スコープ外 (混ぜない)

- LP の教育セクション化 (既存の FAQ と PRIVACY の導線で足りるため見送り)。
- 削除前サマリーへの集計の再利用は次 plan (ゴミ箱化) で行う。
- 既存の Claude (default) のサイズ集計。
