    環境を分けて、混ぜない。

    既存の Claude はそのまま。そこを基準に、何を引き継ぎ何を分けるかを決めて、新しい環境を作ります。

    1. 1環境を作る を押す
    2. 2すべて分ける か 設定だけ引き継ぐ を選ぶ
    3. 3メニューバーから切り替える

    いまの環境はそのまま残ります

    あなたが普段使っている Claudeデスクトップアプリと Claude Code の環境は「既存の Claude」として保護され、設定・履歴・ログインは変更されません。

    環境を切り替えて使えます

    「利用中」はメニューバーがいま対象にしている環境です。別の環境に切り替えるには、先に起動中の Claude を終了してから「この環境で Claude を起動」を押します。設定の衝突を防ぐため、複数の環境の Claude を同時には開けません。

    ターミナル（Claude Code）も同じ環境で使えます

    CSW から開いた Claudeデスクトップアプリの中で開くターミナルは、最初からこの環境です。そのまま claude で始められます。別に開く iTerm2 などのターミナルでこの環境を使うときだけ、次のコマンドを実行します（そのタブだけに適用され、普段の環境には影響しません）。

    eval $(csw env <環境名>)
