    環境を分けて、混ぜない。

    いまの Claude はそのまま。仕事用や検証用に、独立した Claude 環境を作れます。

    1. 1環境を作る を押す
    2. 2まっさらに分ける か 設定だけ引き継ぐ を選ぶ
    3. 3メニューバーから切り替える

    いまの環境はそのまま残ります

    あなたが普段使っている Claudeデスクトップアプリと Claude Code の環境は「いま使っている Claude」として保護され、設定・履歴・ログインは変更されません。

    ターミナルでも同じ環境を使えます

    Claude Code（CLI）で使うには、ターミナルで次のコマンドを実行します。このタブだけに適用され、普段の環境には影響しません。

    eval $(csw env "<名前>")
