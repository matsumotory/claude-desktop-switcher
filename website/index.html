Integrated Claude Desktop
& Claude Code Control

Switch profiles from the menu bar to manage your Claude Desktop data directory and Claude Code environment together. Keep both tools aligned under the exact same profile.

Customizable Separation

Define exactly what to share and what to isolate. For example, you can completely separate your login sessions and past chat histories between 'Work' and 'Personal' profiles, ensuring your data never crosses over.

Zero System Impact

Your Mac's system settings are never overwritten. Active profiles are applied strictly locally—only to the Claude Desktop app launched via this tool or your specific terminal tab—ensuring it safely coexists with your usual development environment.

Integration & Workflow

Build completely isolated profiles without disrupting your existing setup.

01

Initial Setup

Simply drop it into your Applications folder and launch. Existing environment variables remain untouched, coexisting safely.

02

Profile Creation

Create independent environments tailored to your needs. Choose between complete isolation (default) or sharing specific configurations.

03

Terminal Integration

Run eval $(csw env) in your terminal. Claude Code instantly operates within the same isolated profile.
